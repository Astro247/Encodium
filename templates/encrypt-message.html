To encrypt a message using the RSA system, it is essential to first transform the textual content into a numerical format. This is because the RSA algorithm operates strictly on integers through mathematical operations involving modular exponentiation.

If the message to be encrypted is a string, the first step is to split it into individual characters. Each character is then converted into its corresponding numerical value using the ASCII table.

Once the numerical representation of the message is obtained, the actual RSA encryption is applied. The encrypted character c is obtained by raising the original character, converted to its ASCII value m, to the power of the public exponent e, and then computing the result modulo n, where n is the product of the two prime numbers used to generate the RSA keys.

$$ c = m^e \:\: mod \:\: n $$

The result c is an encrypted number representing the original character. This process is repeated for every character in the message.
