To decrypt a message, which has been encrypted using a specific public key (n, e), it is necessary to possess the corresponding private key (n, d).

The original character, converted into its ASCII representation m, is obtained by raising the encrypted character c to the power of d (the private exponent), and then computing the result modulo n.

This operation can be expressed mathematically as:

$$ m = c^d \:\: mod \:\: n $$

After repeating the following operation for each number, the result is a sequence of ASCII values that, once converted into their corresponding characters, reconstructs the original human-readable message.
